# abgabe_quaranteam

Im folgenden sind enthalten:

1. Das Frontend in Form von React Native Expo:
    Einfach
    - *npm install*
    - *expo start*
    ausführen.
2. Das Backend in Form von Flask:
    - flask_run.bat ausführen 
    oder
    - Flask App festlegen & *flask run* ausführen
3. Postgresql (SQL Alchemy als ORM)
    Datenbank aus dem Backend migrieren mit:
    init_db.bat
