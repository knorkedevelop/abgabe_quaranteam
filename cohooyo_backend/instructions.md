##Vorgehensweise
    1. Für die Erstinstallation bitte:
        - Mit **pip freeze >requirements.txt** Datei erstellen
        - Und mit **pip install -r requirements.txt** alle Requierments nachinstallieren
    2. Für die initialisierung der DB **init_db.bat** ausführen
    3. Zum Starten **flask_run.bat** benutzen
    4. Virtual Environment **py -3 -m venv venv** **venv\Scripts\activate**
    5. Sprachpackt download **python -m spacy download de_core_news_lg**
